# DataPreprocessor

Реализация класса `DataPreprocessor` для автоматической очистки и подготовки табличных данных (pandas DataFrame). Проект выполнен в рамках тестового задания на стажировку.

## Описание

Класс принимает на вход "сырой" датафрейм и выполняет последовательный пайплайн обработки данных через метод `fit_transform`. 

Основные возможности:
1.  **Обработка пропусков:** 
    *   Удаляет столбцы, где количество пропусков превышает заданный порог (`threshold`).
    *   Оставшиеся пропуски заполняет: для чисел — средним значением, для категорий — модой.
2.  **Кодирование категорий:** 
    *   Применяет One-Hot Encoding ко всем строковым колонкам.
3.  **Нормализация чисел:** 
    *   Поддерживает два метода: Min-Max нормализация (по умолчанию) и Стандартизация (Z-score).
4.  **История:** 
    *   Класс запоминает, какие столбцы были удалены, какие новые добавлены и какие параметры нормализации использовались.

## Структура проекта

*   `data_preprocessor.py` — файл с основным классом `DataPreprocessor`.
*   `demo.ipynb` — Jupyter Notebook с демонстрацией работы на примере датасета Titanic.
